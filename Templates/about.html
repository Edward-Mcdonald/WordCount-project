This is a word counter webpage that utilises Django principles, and integrates HTML, Python and some JavaScript programming.

It is an example webpage (to practise programming) that takes an string input provided by the user e.g. song lyrics, a speech, a sentence, etc. and returns the total word count. It also returns the words and the number of times they appear in the input, starting with the highest and descending order.
